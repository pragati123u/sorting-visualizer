About Bubble Sort

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

How to Use

  • Adjust the Array Size slider to change the number of elements
  • Adjust the Sorting Speed slider to control the visualization speed
  • Select a sorting algorithm from the dropdown menu
  • Click Generate New Array to create a new random array
  • Click Start Sorting to begin the visualization
  • Observe the statistics (comparisons, swaps, time) during the sorting process
